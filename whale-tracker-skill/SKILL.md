# Whale Tracker - 大口投資家動向分析スキル

## このスキルの目的

機関投資家(ヘッジファンド・年金基金・投資銀行)、アクティビスト、企業役員(インサイダー)などのスマートマネーが「今、どの銘柄を買い集めているか」を、複数の公開データソースから多角的に検出してスコア化する。

## 5つのシグナル(米国市場)

### ① Form 4 (インサイダー取引) — 最強シグナル
- **データソース**: SEC EDGAR (`https://data.sec.gov/submissions/CIK{CIK}.json`)
- **見るべきポイント**:
  - **取引コードP** = 自腹のオープンマーケット買付(最重要)
  - 直近30日のクラスターバイ(複数役員が同期間に買い)
  - CEO/CFOの$500K+の単独買い
  - 10%以上保有者の追加買い
- **重み**: 1.5x

### ② 13D/13G (5%以上大量保有)
- **データソース**: SEC EDGAR full-text search
- **見るべきポイント**:
  - 13D = 経営介入意思あり(より強気)
  - アクティビスト名 (Pershing Square, Elliott, Engine No.1, Icahn等) の有無
  - Amendment(変更報告)で保有比率増加
- **重み**: 1.3x

### ③ オプション異常出来高
- **データソース**: Yahoo Finance Options Chain
- **見るべきポイント**:
  - スイープ(複数取引所分散の大口注文)
  - Vol/OI比 > 5x = 新規ポジション急増
  - コールプレミアム合計 > $500K
  - Call/Put比 >= 2.0 (強気)
- **重み**: 1.4x

### ④ ダークプール出来高
- **データソース**: FINRA Daily Short Sale Volume Files (`https://cdn.finra.org/equity/regsho/daily/`)
- **見るべきポイント**:
  - Off-Exchange ショート比率 >= 60% = 機関のヘッジ付きロング示唆
  - 出来高絶対量の異常
- **重み**: 1.0x

### ⑤ 13F (機関投資家四半期)
- **データソース**: SEC EDGAR (45日遅延)
- **見るべきポイント**:
  - スーパーインベスター(Buffett, Burry, Druckenmiller等)の新規買い
  - 複数ファンドのクラスター現象
- **重み**: 0.8x

## 3つのシグナル(日本市場 — 米国と異なる)

### ① EDINET 大量保有報告 (5%超)
- **データソース**: EDINET API v2 (`https://api.edinet-fsa.go.jp/api/v2/documents.json`)
- **doc_type**: 350(初回)/360(変更)/370(訂正)
- **見るべきポイント**:
  - 著名アクティビスト(オアシス、エフィッシモ、村上系)
  - BlackRock/Vanguard等のメガ機関の初回提出
  - 5営業日以内開示 = 米国13D/G(10日)より早い
- **重み**: 1.5x

### ② 出来高急増 (ToSTNeT代理指標)
- **データソース**: Yahoo!ファイナンス株価ヒストリー
- **見るべきポイント**:
  - 出来高3倍以上 + 価格2%以上上昇 = 機関本気買い
  - 出来高急増 + 価格横ばい = 吸収局面(ブレイク前)
- **重み**: 1.4x

### ③ JPX空売り残高 (0.5%超)
- **データソース**: JPX 空売り残高ページ(日次Excel)
- **見るべきポイント**:
  - 残高合計5%以上 = スクイーズ余地大
  - **米国SI(月2回)より高頻度**で日本市場の優位データ
- **重み**: 1.0x

## スコアリング

```
US Whale Score = (insider * 1.5) + (13d * 1.3) + (options * 1.4) + (darkpool * 1.0) + (13f * 0.8)
JP Whale Score = (holdings * 1.5) + (volume * 1.4) + (short * 1.0)
```

範囲: 0〜100に正規化

| スコア | 分類 | アクション |
|---|---|---|
| 80-100 | 🔥 STRONG | 即監視リスト入り、寄り付き買い候補 |
| 65-79 | ⭐ WATCH | 寄り付き様子見後エントリー検討 |
| 50-64 | 👀 MONITOR | 追加シグナル待ち |
| 0-49 | LOW | スルー |

## 使い方

### ユーザーが「大口投資家の動きを調べて」と聞いた場合

1. **対象市場を確認**(米国株 or 日本株、または両方)
2. **対象銘柄を確認**(具体的ティッカー or スクリーニング)
3. **本ツールで`fetch_us.py`または`fetch_jp.py`を実行**(または保存済みデータがあればそれを参照)
4. **結果を提示**:
   - スコア順に並べる
   - 各シグナルの根拠を明示
   - 注意事項(データ遅延)を添える

### 既存スキル `daytrade-scanner` との連携

```
[whale-tracker]    → Whale Score 65+ の候補銘柄抽出
       ↓
[daytrade-scanner] → 寄り付きの大陽線判定(EPS, 出来高, ギャップ)
       ↓
   エントリー候補
```

**勝率最大化の組み合わせ**: Whale Score 80+ × 寄付き大陽線シグナル × 高短期金利銘柄

## ペナルティ要素(警戒シグナル)

- 大口インサイダー売り(CEO/CFO、$1M以上): -20点
- プットフロー優位(コール対比2倍以上): -15点
- 13Fでメガファンドが大量売却: -10点
- 出来高急増 + 価格大幅下落: -6点(日本)

## 限界と注意

- **データ遅延**: Form 4(2日) / EDINET(5日) / 13F(45日) / FINRA(T+1) / 空売り残高(T+2)
- **インサイダー売却**は税金・分散目的の場合があり一概に弱気とは限らない
- **45日遅れの13F**は既にクローズされている可能性
- **大量保有報告 ≠ 即上昇**: パッシブ機関の場合は値動きに繋がらないことも
- 投資判断は最終的にユーザー責任
