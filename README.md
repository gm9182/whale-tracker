# 🐋 Whale Tracker v3.0 — 大口投資家動向分析ツール

**スマートマネー(機関投資家・アクティビスト・インサイダー)の足跡を、無料公開データから多角的に検出するスマホ最適化ツール。日米両市場対応・GitHub Actionsで完全自動更新。**

## 🆕 v3.0 新機能

### ✨ S級機能(勝率直結)
1. **TDnet適時開示の全件取込**(日本) — 上方修正・自社株買い・業務提携・株式分割・配当変更等を即時検出
2. **信用取引残高の急変分析**(日本) — 信用倍率<1.0 + 売残急増 = 踏み上げ余地検出
3. **米国オプションスイープ詳細化** — 複数満期スプレッド・ブロック検出・緊急性タグ付け

### ✨ A級機能(UI/UX)
4. **⚡ MOVERSフィルタ** — スコアが前回より±10以上変化した銘柄をハイライト
5. **★ お気に入り機能** — LocalStorageで端末ごとに保存
6. **市場時計** — JPX/NYSEの営業状況をリアルタイム表示
7. **5分自動再読み込み** — 新データが来たら勝手に反映

## 🚀 セットアップ(15分・PC作業)

1. リポジトリを作成・push
2. GitHub上でSettings → Pages → Source: "GitHub Actions"
3. Actions タブ → Run workflow(初回のみ手動起動)
4. 約20分後 `https://<username>.github.io/<repo>/` でアクセス可
5. iPhoneでURLを開く → 共有 → ホーム画面に追加

## 💡 プロのトレード戦略

### 寄り付き勝率最大化セットアップ (日本株)
1. **前夜21:00以降**: TDnet全件取込を確認 — 上方修正・自社株買い銘柄を抽出
2. **MOVERSタブ** でスコア急変銘柄 (EDINET + 出来高 + TDnet全部入りが理想)
3. **既存 daytrade-scanner** で寄り付き判定 (EPS、出来高、ギャップ)
4. **Whale Score 80+ × 大陽線判定 ◎** = 最強候補

### 米国株スイープ追随戦略
1. **市場前(JST 22:00頃)**: BLOCKタグ付きコールを確認
2. **複数満期スプレッド検出** = 機関の本格ポジション構築
3. **緊急コール集中(直近1時間)** = 情報優位の可能性

### MOVERSフィルタの活用
- **+15以上** = 機関の新規エントリー検出
- **-15以下** = ポジション解消検出 → ロングは様子見

## 🕐 自動更新スケジュール

GitHub Actions が市場時間帯に集中:
- **JST 9:00-15:00**: 1時間ごと(日本市場)
- **JST 22:00-翌5:00**: 1時間ごと(米国市場)
- **JST 6:00-7:00**: 寄付き前データ反映

## ⚠️ 注意事項

- **TDnet非公式API**: yanoshin.jp提供 — 突発停止の可能性あり(代替: JPX有料API)
- **信用残データ**: softhompo.a.la9.jp提供 — URL構造変更時はパース失敗あり
- **データ遅延**: Form 4(2日)、EDINET(5日)、TDnet(リアルタイム)、空売り残高(T+2)、信用残(T+2)
- **本ツールは情報提供のみ**で投資助言ではありません
