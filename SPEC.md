# 大口投資家(スマートマネー)動向分析ツール 設計書

**バージョン:** 1.0
**作成日:** 2026年5月3日
**対象ユーザー:** 米国株デイトレーダー / スイングトレーダー

---

## 1. 目的と狙い

### 1.1 ツールが解決する課題
個人投資家が機関投資家(ヘッジファンド、投資銀行、年金基金など)の動きを察知することは難しい。彼らは:

- **ダークプール**(非公開取引所)で大口注文を執行し、市場参加者に気づかれないようにする
- **オプションのスイープ取引**で複数取引所に分散注文し、痕跡を消す
- 売買成立後**2営業日以内のSEC報告**で初めて公になる(インサイダー取引)
- 四半期報告(13F)は**最大45日遅れ**で公開される

→ これらの「足跡」を**無料の公開データ**から逆算的に検出することで、大口の建玉方向を推定する。

### 1.2 ツールのゴール
**「機関投資家が今、どの銘柄を買い集めているか」を多角的シグナルから定量スコア化し、デイトレード/スイング候補を提示する。**

## 2. 大口投資家の判断基準(プロが見ている5つのシグナル)

### シグナル①:インサイダー取引(SEC Form 4) — 最強シグナル

**根拠:** 企業の役員・取締役・10%以上保有者は、自社株の売買から**2営業日以内**にSECにForm 4を提出する義務がある。学術研究では、**インサイダーの買い(P=Purchase)は今後30~90日のリターンを予測する**とされている。

**プロの見方:**
| 取引コード | 意味 | 重要度 |
|---|---|---|
| **P** | オープンマーケットでの買付 | ★★★★★ 最重要(自腹で買う=本気度高) |
| S | オープンマーケットでの売却 | ★★ ノイズ多(税金/分散目的の場合あり) |
| A | 報酬としての株式付与 | ★ ほぼ無視 |
| M | オプション行使 | ★★ 文脈依存 |
| F | 税金支払いのための株式売却 | ★ 無視 |

**ツールでの判定ロジック:**
- 直近30日以内のクラスターバイ(複数役員が同時期に買付): **+25点**
- CEO/CFO単独でも$500K以上の買付: **+15点**
- 10%以上保有者の買い増し: **+15点**
- 役員の売却が複数: **-10点(警戒)**

**データソース:** SEC EDGAR(無料、公式)
- URL: `https://www.sec.gov/cgi-bin/browse-edgar?action=getcompany&type=4`
- API: `https://data.sec.gov/submissions/CIK{CIK}.json`

---

### シグナル②:13D/13G(5%以上の大量保有報告)

**根拠:** ある投資家(ヘッジファンドや個人)が企業株式の**5%以上**を取得した場合、10営業日以内に13D(積極的)または13G(消極的)を提出する。**13Dは経営介入の意思**を示すため特に重要。

**プロの見方:**
- **13D提出 + 著名アクティビスト名(Pershing Square, Elliott, Engine No.1等)** = 経営改革期待で爆騰の可能性
- **13G** = 受動的だが、大口資金が長期保有する意思の表れ

**ツールでの判定ロジック:**
- 著名アクティビストファンドの13D提出: **+30点**
- 一般機関投資家の13G提出(新規): **+10点**
- 既存ポジションの追加(Amendment): **+5点**

**データソース:** SEC EDGAR Form 13D/13G

---

### シグナル③:オプション異常出来高(Unusual Options Activity)

**根拠:** オプションは少額でレバレッジが効くため、機関投資家が**情報優位性を金に換える時の常套手段**。学術論文「Informed Trading in the Option Market」は、満期間近のOTMコールに大口注文が出た翌日~1週間以内に対象株が大きく動くことを実証している。

**プロの見方(4つのパターン):**

| パターン | 説明 | シグナル強度 |
|---|---|---|
| **スイープ** | 複数取引所に分散して数十万ドル~数百万ドル規模を同時執行 | ★★★★★(緊急性=情報あり) |
| **ブロック** | 単一取引所で大口注文(10,000枚以上) | ★★★★ |
| **ボリューム >> OI** | 出来高が建玉の数倍以上(新規ポジション) | ★★★★ |
| **OTMコール大量買い** | 権利行使価格が現値より10%以上上(投機色強) | ★★★ |

**ツールでの判定ロジック:**
- コールスイープ($500K以上、満期45日以内): **+25点**
- Vol/OI比率 > 5.0 + コール優位: **+20点**
- プットスイープが多い場合: **-15点(下方向のヘッジまたは弱気)**

**データソース:**
- 無料: Barchart Unusual Options Activity(25分遅延、無料)
- 無料: Yahoo Finance Options Chain(フィルタリング自前)
- CBOE公開データ

---

### シグナル④:ダークプール出来高(Off-Exchange Volume)

**根拠:** 米国株の**約40~45%が市場外(ダークプール)で取引**されている。機関投資家は値段を動かさずに大量の株を仕込みたい時、ダークプールを使う。**ダークプールでの買い集めは、しばしば公開市場での値上がりに先行する。**

**プロの見方:**
- **ショートボリューム比率(Off-Exchange Short Volume Ratio)**:50%以上 = ヘッジ目的の機関買いの可能性(空売りに見えても実は機関のヘッジ付きロング)
- **Off-Exchange Total Volume**:平均比200%以上 = ダークプールでの大口集約の兆候
- **DPI(Dark Pool Index)**:45以上 = ブル、35以下 = ベア

**ツールでの判定ロジック:**
- ダークプール出来高が20日平均比2倍以上: **+15点**
- ダークプールショート比率 > 60%: **+10点(機関ヘッジロング示唆)**
- ダークプール出来高で価格が動かない(吸収局面): **+10点**

**データソース:** FINRA Daily Short Sale Volume Files(無料、日次更新)
- URL: `https://www.finra.org/finra-data/browse-catalog/short-sale-volume-data/daily-short-sale-volume-files`
- 形式: テキストファイル(パイプ区切り)

---

### シグナル⑤:13F(機関投資家四半期報告)

**根拠:** 運用残高1億ドル以上の機関投資家は、四半期末から45日以内に保有銘柄をSECに報告する義務がある。**45日遅れだが、複数のスマートマネーが同時に同じ銘柄を集めている動きはトレンドの先行指標**となる。

**プロの見方:**
- **超有名ファンドの新規ポジション** (Buffett, Burry, Ackman, Druckenmiller, Tepper等)
- **クラスター現象**:同四半期に5社以上の有名ファンドが同銘柄を新規購入
- **既存ポジションの大幅増加**(50%以上の追加)

**ツールでの判定ロジック:**
- スーパーインベスター3社以上が同時新規買い: **+20点**
- 1社のみ(ただし著名)新規買い: **+10点**
- 機関投資家全体の保有比率が前四半期比+5%以上: **+5点**

**データソース:** SEC EDGAR Form 13F-HR

## 3. 統合スコアリング・モデル

### 3.1 総合スコア計算式

```
Whale Score = (W1 × Insider) + (W2 × 13D/G) + (W3 × Options) + (W4 × DarkPool) + (W5 × 13F) - Penalties
```

| 要素 | 重み(W) | 最大点数 |
|---|---|---|
| W1: インサイダー取引(直近性が高い) | 1.5 | 60 |
| W2: 13D/13G(明示的シグナル) | 1.3 | 30 |
| W3: オプションフロー(短期的) | 1.4 | 45 |
| W4: ダークプール(継続的) | 1.0 | 35 |
| W5: 13F(中長期トレンド) | 0.8 | 25 |

**最終スコア範囲: 0~100**

### 3.2 シグナル分類

| スコア | 分類 | アクション |
|---|---|---|
| 80-100 | 🔥 **STRONG BUY** | 複数シグナル同時点灯、即監視リスト入り |
| 65-79 | ⭐ **BUY WATCH** | 寄り付き様子見後エントリー検討 |
| 50-64 | 👀 **MONITOR** | 追加シグナルを待つ |
| 0-49 | — | スルー |

### 3.3 ペナルティ要素

- 大口インサイダー売り(CEO/CFO、$1M以上): **-20点**
- プットフロー優位(コール対比2倍以上): **-15点**
- 13Fでメガファンドが大量売却: **-10点**
- ダークプールが急激な売り集中: **-10点**

## 4. 既存スキル(daytrade-scanner)との連携

ユーザーは既にデイトレ用スキャナーを持っているため、本ツールは**「銘柄選定の上流工程」**として機能する:

```
[本ツール] → 大口シグナル検出 → 候補銘柄リスト
    ↓
[daytrade-scanner] → 寄り付きの大陽線判定 → エントリー
```

**連携ポイント:**
- 本ツールで検出した銘柄に対し、daytrade-scannerの判断基準(EPS, 出来高, 寄り付きギャップ等)を適用
- ms9182様の検証済みルール(2026/05/01結果)との整合性チェック
- 高スコア銘柄を翌日寄り付き候補として優先順位付け

## 5. システム構成

### 5.1 アーキテクチャ
```
┌─────────────────────────────────────┐
│   スマホ(ブラウザ)                  │
│   ↓ アクセス                         │
│ [HTMLダッシュボード]                 │
│   ↓ 表示                             │
│ ・銘柄スコア一覧                     │
│ ・各シグナルの詳細                   │
│ ・チャート/データ可視化              │
└─────────────────────────────────────┘
       ↑ データ取得
┌─────────────────────────────────────┐
│ データソース(全て無料)              │
├─────────────────────────────────────┤
│ ・SEC EDGAR (Form 4, 13D/G, 13F)    │
│ ・FINRA (ダークプール、ショート出来高) │
│ ・Yahoo Finance (株価、オプション)   │
│ ・Barchart (オプション異常出来高)    │
└─────────────────────────────────────┘
```

### 5.2 動作モード

**モード1: ライブ検索モード**
- ユーザーが銘柄ティッカーを入力 → 全シグナルを集計してスコア表示

**モード2: スキャンモード(デモデータ含む)**
- 注目銘柄リスト(S&P500等)から高スコア銘柄を自動抽出
- ※実データ取得は無料APIのレート制限のため、デモデータ + 主要シグナルのみライブ取得のハイブリッド

**モード3: シグナル詳細モード**
- 個別銘柄をタップ → 各シグナルの内訳と根拠データを表示

## 6. 注意事項と限界

### 6.1 データの遅延
| データ種別 | 遅延 |
|---|---|
| Form 4(インサイダー) | 取引から最大2営業日 |
| 13D/13G | 取得から最大10営業日 |
| 13F | 四半期末から最大45日 |
| FINRA ダークプール | T+1日(日次更新) |
| オプション(無料源) | 25~30分遅延 |

### 6.2 シグナルの限界
- **インサイダー売り**は税金/分散目的の場合があり、即弱気とは限らない
- **13F**は45日遅れで、すでにクローズされている可能性あり
- **ダークプール**だけでは方向性(買い/売り)が判別できないため、価格との組み合わせ必須
- **オプションスイープ**は機関のヘッジ目的の可能性もあるため、株価動向と併せて判断

### 6.3 法的事項
- 本ツールは**情報提供目的**のみであり、投資助言ではない
- 全シグナルを満たしても株価上昇は保証されない
- 投資判断の最終責任はユーザーにある
