# 日本市場版 Whale Tracker 設計書

**米国版との根本的な違い、および日本市場で「大口の足跡」を読むための独自ロジックを解説。**

---

## 1. なぜ米国と同じ手法が使えないか

| 項目 | 米国 | 日本 |
|---|---|---|
| 役員売買のリアルタイム開示 | Form 4 (2日以内) | 役員報告書(数日〜月単位、遅延) |
| 5%以上の保有報告 | 13D/G (10日以内) | EDINET大量保有報告(5営業日以内) ✓ |
| 機関の四半期報告 | 13F (45日後) | 機関投資家の自主開示のみ(義務なし) |
| ダークプール出来高 | FINRA毎日公開 | ToSTNeT集計は限定的 |
| 個別株オプションフロー | 流動性高、機関活用多 | 流動性低、機関は先物・指数オプション中心 |
| 空売り残高 | 月2回(SI) | 毎日0.5%以上公開 ✓ (米国より優位な点) |

**結論**: 日本ではインサイダーよりも「**EDINETに記録される機関の意思表示**」と「**毎日公開される空売り残高の動き**」を中心にシグナル化するのが最適。

## 2. 日本版で採用したシグナル

### シグナル①: EDINET 大量保有報告書 — 最重要

**根拠:**
- 上場会社の発行済株式総数の**5%以上を保有した者**は5営業日以内に金融庁EDINETに「大量保有報告書」を提出する義務がある
- 1%以上の保有割合変化があれば「**変更報告書**」を提出
- 米国の13D/Gに相当するが、**5営業日以内 = 米国(10日)より早い**ため、シグナル鮮度は日本の方が高い

**プロの見方:**
| 報告種別 | 重み |
|---|---|
| 著名アクティビスト(オアシス、エフィッシモ、村上系)の初回提出 | ★★★★★ |
| BlackRock/Vanguardなど巨大機関の初回提出 | ★★★★ |
| 一般機関の初回提出 | ★★★ |
| 既存ポジションの追加(変更報告) | ★★ |
| 保有比率減少(変更報告) | ✕(ペナルティ要素) |

**ツールでの判定:**
- 初回大量保有提出: **+25点**
- 著名アクティビストの提出: **+15点追加**
- 変更報告(増加方向): **+10点**
- 上限: 40点

**データ取得:**
- EDINET API v2 (`https://api.edinet-fsa.go.jp/api/v2/`)
- `/documents.json?date=YYYY-MM-DD&type=2` で日次提出書類一覧
- doc_type 350(初回)/360(変更)/370(訂正)で識別

---

### シグナル②: 出来高急増(ToSTNeT代理指標) — 短期勝負の主軸

**根拠:**
- 日本では**ToSTNeT(立会外取引)の銘柄別出来高が個人入手しにくい**(JPXは集計値しか公開しない)
- そのため「**当日出来高 ÷ 過去20日平均**」を機関の集中買いの代理指標として使用
- 価格上昇を伴う出来高急増 = **機関の指値での買い集め**を示唆

**プロの見方:**
- 出来高3倍以上 + 価格2%以上上昇 = 機関の本気買い
- 出来高急増 + 価格は変わらず = 機関の吸収局面(ブレイク前夜)
- 出来高急増 + 価格大幅下落 = 機関の利確/逃げ

**ツールでの判定:**
- 出来高比3倍以上: **+20点**
- 出来高比2-3倍: **+12点**
- 出来高比1.5-2倍: **+6点**
- + 価格+2%以上: **+8点追加**
- 価格-2%以下: **-6点(警戒)**
- 上限: 35点

**データ取得:**
- Yahoo!ファイナンス株価ヒストリーページから時系列出来高をスクレイプ
- 銘柄コード `XXXX.T` 形式で取得

---

### シグナル③: JPX 空売り残高(0.5%以上) — スクイーズ余地の検出

**根拠:**
- 東証は「有価証券の取引等の規制に関する内閣府令」に基づき、**残高割合0.5%以上の空売りポジションを毎営業日17時頃公表**
- 米国のSI(月2回)より**遥かに高頻度** = 日本の優位データ
- 残高が大きい銘柄は、**ポジティブニュース時にショートカバー(踏み上げ)**で爆発的上昇が起こる

**プロの見方:**
- 空売り残高合計5%以上 → スクイーズ余地大
- 著名ヘッジファンドの新規空売り → 警戒(売り意図ある可能性)
- 空売り残高の急減 → 既にカバー進行中(上昇開始)

**ツールでの判定:**
- 残高合計5%以上: **+12点**
- 残高合計3-5%: **+8点**
- 残高合計1-3%: **+4点**
- 上限: 12点

**データ取得:**
- JPX 「空売り残高に関する情報」ページから日次Excelをダウンロード
- パースして銘柄別残高合計を集計

## 3. 統合スコアリング(日本版)

```
JP Whale Score = (W1 × Holdings) + (W2 × Volume) + (W3 × Short)
              W1=1.5, W2=1.4, W3=1.0
```

最大点数: 40×1.5 + 35×1.4 + 12×1.0 = 121 → **0〜100に正規化**

| スコア | 分類 | 意味 |
|---|---|---|
| 80-100 | 🔥 STRONG | 大量保有報告 + 出来高急増 + 上昇継続の3拍子揃い |
| 65-79 | ⭐ WATCH | 2シグナル点灯、寄り付き様子見後エントリー |
| 50-64 | 👀 MONITOR | 1シグナルのみ、追加シグナル待ち |

## 4. 日本市場特有の使い方Tips

### A. 寄り付きの監視ポイント
- **9:00 寄り前気配**: STRONG銘柄の気配を確認(大引け終値+5%以上は買い集中)
- **9:30以降**: 出来高急増+VWAP上で推移していれば追随OK
- **10:30 ザラ場一服**: 続伸銘柄は午後の機関買い継続の可能性

### B. 大量保有報告の発表タイミング
- EDINETへの提出は **平日 9:00 ~ 17:15**
- 著名ファンドの提出は**寄り後 ~ 14:00頃に集中**することが多い
- = 翌日の寄り付きが最も反応が大きい(夜間に情報拡散)

### C. 信用残との組み合わせ(将来拡張案)
JPXは毎週金曜に「個別銘柄信用取引残高表」を公表:
- **信用買残急減 + 出来高急増 + Whale Score高** = 強烈な踏み上げ候補

### D. デイトレード判断との連携
既存の `daytrade-scanner` スキルとの統合:
- Whale Score 65+ かつ 寄り前のEPS良好 → エントリー候補
- Whale Score 80+ なら寄り付き直後の買いも視野
- Whale Score 50以下 → スルー推奨

## 5. 限界と注意点

### A. データの遅延
- EDINET大量保有: 取得から最大5営業日遅延
- 空売り残高: T+2日(2営業日前のスナップショット)
- 出来高は日中分(リアルタイムに近いが取引中の途中値)

### B. シグナルの限界
- **大量保有報告 ≠ 即上昇**: パッシブ運用の機関の場合は値動きに繋がらないことも
- **空売り残高大 ≠ 必ず踏み上げ**: 業績悪化の根拠があれば逆に下落継続
- **出来高急増 ≠ 機関の買い**: 仕手筋・個人投機の可能性も(信用残・呼値併用で判別)

### C. 業種別の特徴
- **半導体株**: 出来高1.5倍程度でもインパクト大(普段の流動性が高いため)
- **中小型グロース**: 出来高2倍以下はノイズ(スイング筋の手仕舞いも多い)
- **メガバンク**: 機関の常連入れ替えがあるため、Whale Score高くてもブレイクには繋がりにくい

## 6. 将来の拡張(優先順位順)

1. **個別銘柄信用残**(週次): 踏み上げ余地の精緻化
2. **裁定残高**: 先物との需給ギャップ検出
3. **TDnet適時開示連携**: 自社株買い・株式分割・上方修正の即時取り込み
4. **時間外取引(PTS)気配**: 寄り付き予想精度向上
5. **業種別相対比較**: 同セクター内でのWhale Score偏差値化
